# Expectation-Maximization: Aplicación en Neurociencias 

Agustin Diaz Barquinero y Ezequiel Pereyra

Este repositorio contiene el material práctico y el código fuente correspondiente a la clase y TP Final sobre el algoritmo **Expectation-Maximization (EM)**.

El objetivo del proyecto es demostrar la aplicación de **Modelos de Mezclas Gaussianas (GMM)** entrenados con EM para resolver un problema real neurociencias: la clasificación de potenciales de acción (spikes) a partir de registros extracelulares obtenidos en ratas en su corteza prefrontal.
